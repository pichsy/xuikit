# xuikit
最喜欢的自定义控件
可结合xskinloader 扩展换肤


### 引入控件
最新版本:  [![](https://jitpack.io/v/pichsy/xuikit.svg)](https://jitpack.io/#pichsy/xuikit)

        
       implementation 'com.github.pichsy:xuikit:1.4'
       

## 用法

    
      
      // 头部导航
      XToolbarLayout
      
      // 常用cells布局
      CommonItemView
      
      // 底部导航
      BottomBarLayout
      
      // 底部导航item
      BottomBarItem
    
    // 弧形头部，或者衔接布局
    HeadArcView
